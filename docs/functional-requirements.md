## Overview

Create a React application and AWS Lambda (and all key AWS infrastructure) to retrieve and display customer data in a paginated format.

This challenge tests your ability to:

- Build scalable front-end interfaces
- Implement efficient data handling
- Work with serverless backends

---

## Requirements

### Frontend (React)

Create a responsive React application with the following features:

- Displays a paginated table or list view of customers retrieved from an API implemented in an AWS Lambda

Each customer record must display:

- Customer ID
- Full Name
- Email
- Registration Date

#### Implementation Requirements

- Implement proper TypeScript types and interfaces
- Include unit tests for critical components
- Use proper state management (Context API or Redux)

---

### Backend (AWS Lambda)

Create a Lambda function that:

- Accepts query parameters for pagination
- Returns properly structured JSON responses
- Includes appropriate error handling
- Implements request validation
- Conforms to REST best practices

You are free to choose the simplest data source for your customer data.

---

- Code quality
- Functionality
- Testing

- Provide a GitHub repository for your implementation

- Maximum 8 hours
